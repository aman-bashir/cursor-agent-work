# VPS Configuration for Deployment
# Copy these values to GitHub Secrets (Settings → Secrets → Actions)
# NEVER commit actual values to git!

# Required Secrets (Add to GitHub):
VPS_HOST=your-vps-ip-address-here
VPS_USER=your-username-here
VPS_PORT=22
VPS_SSH_KEY=<paste-entire-private-key-content-here>

# How to get SSH Private Key:
# 1. SSH to your VPS
# 2. Run: cat ~/.ssh/id_rsa
# 3. Copy entire output (including BEGIN and END lines)
# 4. Paste in GitHub Secret: VPS_SSH_KEY
